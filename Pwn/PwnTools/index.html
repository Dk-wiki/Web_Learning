    PwnTools¶

    安装¶

    pip install pwntools

    简单使用¶

    使用procrss接口启动程序¶

    from pwn import * 
p=process('./a.out')                    #执行可执行文件
p=process(['cat','/flag'])          #执行命令 cat /flag
p=process("./a.out",env={'LD_PRELOAD':'./libc.so.6'})       #指定环境变量文件

    当程序运行与远程使用remote¶

    from pwn import * 
p=remote(host,port)

    使用process或者remote获得一个对象后¶

    send(string)                                    #发送string
sendline(string)                            #发送string + 换行符
sendafter(magic,string)             #接收到magic之后 发送string
sendlineafter(magic,string)     #接收到magic之后 发送string + 换行符
recvn(N)                                            #接受N个字节
recvnline(N)                                    #接受一行输出
recvnlines(N)                                   #接受N行输出
recvuntil(magic)                            #一直接受到magic字符串为止
